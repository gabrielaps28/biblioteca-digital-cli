#  Sabor Express — Sistema de Restaurantes (CLI)

Projeto em Python que simula um sistema simples de gerenciamento de restaurantes via terminal. O programa apresenta um menu interativo que permite cadastrar, listar e ativar restaurantes.

##  Funcionalidades

* Exibição do nome do sistema
* Menu interativo no terminal
* Cadastro de restaurantes *(em desenvolvimento)*
* Listagem de restaurantes *(em desenvolvimento)*
* Ativação de restaurantes *(em desenvolvimento)*
* Tratamento de opções inválidas

## ▶ Como executar

1. Certifique-se de ter o Python instalado (versão 3.x).
2. Clone o repositório:

```
git clone https://github.com/gabrielaps28/RestoManager_python.git
```

3. Entre na pasta do projeto:

```
cd RestoManager_python
```

4. Execute o programa:

```
python nome_do_arquivo.py
```
